Information about the data

This is a dataset of movies with “summer” in their title originating from the Internet Movie Database. The variables used in this analysis include the

  • title_type character variable which refers to the type/format of the title (movie, video, or tvMovie)

  • runtime_minutes refers to the primary runtime of the title, in minutes

  • genres character variable includes up to three genres associated with the title

  • average_rating includes weighted average of all the individual user ratings on IMDb

  • num_votes refers to number of votes the title has received on IMDb (titles with fewer than 10 votes were not included in this dataset)

Creating a new std. dev function

newsd <- function(x){
    n <- length(x)
    mean_val <- sum(x) / n
    stddev <- sqrt(sum((x-mean_val)^2)/(n-1))
    return(stddev)
}
